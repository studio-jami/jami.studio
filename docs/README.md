# Documentation

This directory holds the durable operating guidance for the `jami.studio` marketing-site repo.

## Map

- `architecture/` - durable product, routing, content, AI-readiness, and deployment shape.
- `decisions/` - durable decisions and rationale.
- `engineering/` - planning, report, docs, and agent standards.
- `operations/` - deployment, domains, analytics, and maintenance runbooks.
- `roadmaps/` - active implementation plans.
- `_legacy/` - retired plans and superseded reports.

## Active plan

The active implementation plan lives under `docs/roadmaps/`; completed plans retire to
`docs/_legacy/roadmaps/`. Session and orchestration guidance lives under `docs/engineering/agents/`.
Design guidelines live in `docs/design/`.
